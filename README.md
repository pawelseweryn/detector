Detector project based on:
- HY-MiniSTM32V
- SEN55
- SCD41

It measures, displays and writes on SD card values:
- PM1.0
- PM2.5
- PM4.0
- PM10
- VOC
- NOC
- CO2
- 2x TEMPERATURE
- 2x RH
